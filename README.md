# Hilmost Digital Labs — Monorepo

Welcome to the official monorepo for **Hilmost Software Corporation (HSC)**. This repository houses our high-precision utility ecosystem, built with Next.js, TypeScript, and Tailwind CSS.

## 🚀 The Ultimate Toolbox
Our flagship product is a comprehensive suite of free digital utilities designed for banking-grade precision and 100% data privacy.

**Live Site:** [https://hilmost-toolbox.hilmost.net](https://hilmost-toolbox.hilmost.net)

### Key Features
- **Money & Tax Tools:** Loan calculators, WACC solvers, and tax estimators.
- **PDF Utilities:** Private, browser-side merging, splitting, and rotating.
- **Unit Converters:** High-precision transformations for length, weight, and temperature.
- **Health & Wellness:** BMI tracking and daily mindfulness resources.
- **Privacy First:** 100% client-side processing — your data never leaves your device.

---

## 🛠 Project Structure
This is a workspace-based monorepo managed with NPM:

- `apps/hilmost_main`: Corporate landing page and software hub ([hilmost.net](https://hilmost.net)).
- `apps/hilmost_toolbox_web`: The Next.js rebuild of the Ultimate Toolbox.
- `apps/hilmost_corporate`: Subsidiary and holding company information.
- `packages/ui`: Shared React components and design tokens.
- `packages/config`: Shared SEO configurations, sitemap logic, and scripts.

---

## 🏗 Development

### Prerequisites
- Node.js 18+
- Firebase CLI (for deployment)

### Setup
1. Clone the repository.
2. Run `npm install` at the root.
3. Run `npm run dev` to start all apps in development mode.
